# MCP Starter for Puch AI

This is a starter template for creating your own Model Context Protocol (MCP) server that works with Puch AI. It comes with ready-to-use tools for job searching and image processing.

---

## What is MCP?

MCP (Model Context Protocol) allows AI assistants like Puch AI to securely connect to external tools and data sources. Think of it as giving your AI assistant extra superpowers without compromising security.
